# GradeMaster API

A minimal backend service for calculating course grades.
It computes weighted averages, determines pass/fail status, and calculates the required final score.

## Live Demo 

- Application site: https://umutalb.github.io/GradeMaster-Frontend/

## The GradeMaster Frontend

This API is used by the GradeMaster frontend application.

- Frontend repository: https://github.com/Umutalb/GradeMaster-Frontend

## How to Use

1. Open the project in Visual Studio 2026.
2. Run the API with .NET 10 (HTTPS enabled).
3. Use any client or the frontend to make requests to the /api/grades endpoint.
